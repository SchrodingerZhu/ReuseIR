Passes

-convert-reuse-ir-to-llvm Â¶

Convert Low-level ReuseIR Operations to LLVM IR

This pass is serves as the last step in the ReuseIR pipeline. It converts all remaining low-level ReuseIR operations to LLVM IR.


### `-reuse-ir-acquire-release-fusion`

_Fuse ReuseIR acquire and release Operations_

This pass is used to fuse acquire and release operations.
### `-reuse-ir-closure-outlining`

_Outlines ReuseIR Closures_

This pass outlines closures and instantiates their virtual tables.
### `-reuse-ir-expand-control-flow`

_Expand structured control flows for ReuseIR operations_

This pass is used to expand control flows for ReuseIR operations.
- reuse_ir.token.ensure
- reuse_ir.token.free (with nullible token)
- reuse_ir.rc.release

#### Options

-outline-nested-release : Outline nested RC release operations

### `-reuse-ir-gen-freezable-vtable`

_Generate vtables for freezable rc objects_

This pass generates vtables for freezable rc objects.
### `-reuse-ir-infer-union-tag`

_Infer union tags for ReuseIR operations_

This pass is used to infer union tags for ReuseIR operations.
### `-reuse-ir-print-reuse-analysis`

_Print Reuse Analysis_

This pass is used to print the reuse analysis information.
### `-reuse-ir-token-reuse`

_Apply reuse analysis to reuse token and recyle unused tokens_

This pass is used to apply reuse analysis to reuse token and recyle unused tokens.
